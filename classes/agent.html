Fuel Documentation

Agent Class

The agent class identifies the platform, browser, robot, or mobile devise of the browsing agent.

accept_charset($charset = 'utf-8')

The accept_charset method tests to see if the agent accepts a particular character set.

Static Yes
Parameters
Param Default Description
$charset 
'utf-8'
 Character set to be tested.
Returns boolean
Example 
Agent::accept_charset('utf-8'); // returns TRUE or FALSE

accept_language($language = 'en')

The accept_language method tests to see if the agent accepts a particular language.

Static Yes
Parameters
Param Default Description
$language 
'en'
 Language abbreviation to be tested.
Returns boolean
Example 
Agent::accept_language('fr'); // returns TRUE or FALSE

browser()

The browser method gives you the friendly name of the agent's browser.

Static Yes
Parameters None
Returns string
Example 
Agent::browser(); // returns "Chrome"

charsets()

The charsets method gives you all the charsets an agent accepts.

Static Yes
Parameters None
Returns array
Example 
Agent::charsets();

// Returns
Array
(
    [0] => iso-8859-1
    [1] => utf-8
)

is_browser($browser = null)

The is_browser method checks to see if the the agent is in the browser(s) you pass in.

Static Yes
Parameters
Param Default Description
$browser Required String or Array of browsers to be tested.
Returns boolean
Example 
Agent::is_browser('Chrome'); // returns TRUE or FALSE

$browsers = array('Firefox', 'Chrome', 'Opera');
Agent::is_browser($browsers); // returns TRUE if one matches - FALSE if none do

is_mobile($mobile = null)

The is_mobile method checks to see if the the agent is in the mobile(s) you pass in.

Static Yes
Parameters
Param Default Description
$mobile Required String or Array of mobiles to be tested.
Returns boolean
Example 
Agent::is_mobile('iphone'); // returns TRUE or FALSE

$mobiles = array('iphone', 'ipad', 'blackberry');
Agent::is_mobile($mobiles); // returns TRUE if one matches - FALSE if none do

is_referral()

The is_referral method lets you know if the user was referred from another page (TRUE) or came directly to the page (FALSE).

Static Yes
Parameters None
Returns boolean
Example 
Agent::is_referral(); // returns TRUE or FALSE

is_robot($robot = null)

The is_robot method checks to see if the the agent is in the robot(s) you pass in.

Static Yes
Parameters
Param Default Description
$robot Required String or Array of robots to be tested.
Returns boolean
Example 
Agent::is_robot('googlebot'); // returns TRUE or FALSE

$robots = array('googlebot', 'msnbot', 'lycos');
Agent::is_robot($robots); // returns TRUE if one matches - FALSE if none do

languages()

The languages method gets all the languages an agent accepts.

Static Yes
Parameters None
Returns array
Example 
Agent::languages();

// Returns
Array

Static Yes
Parameters None

Static Yes
Parameters None
Returns string

robot()

The robot method passes back the name of the agent if it is a robot.

Static Yes
Parameters None
Returns string
Example 
Agent::robot(); // returns "Googlebot"
