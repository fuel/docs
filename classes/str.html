Str Class

String handling with encoding support

truncate($string, $limit, $continuation = '...', $is_html = false)

Truncates a string to the given length.

Static Yes
Parameters
Param Default Description
$string 
required
 The string to truncate.
$limit 
true
 The number of characters to truncate too.
$continuation 
true
 The string to use to denote it was truncated.
$is_html 
false
 Whether the string has HTML.
Returns The truncated string
Example 
//returns "This is a ..."
echo Str::truncate('This is a String', 10);

increment($str, $first = 1)

Add's _1 to a string or increment the ending number to allow _2, _3, etc.

Static Yes
Parameters
Param Default Description
$string 
required
 The string to increment.
$first 
1
 The number to append to the string.
Returns The incremented string
Example 
//returns "This is a String_1"
Str::increment('This is a String');

//returns "This is a String_2"
Str::increment('This is a String', 2);
