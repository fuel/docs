サーバ

oil サーバを実行すれば、フルスペックの Web サーバをインストールする必要がありません! それは、PHP 5.4 の組み込み Web サーバを使います。コマンドラインからサーバを立ち上げ、指定したホスト名とポート番号 (デフォルトでは http://localhost:8000) でブラウザからアクセスしてください。 

$ php oil server
Listening on http://localhost:8000
Document root is public/
Press Ctrl-C to quit.

使い終わったら、Ctrl-C でサーバを終了します。
