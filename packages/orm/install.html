Installation

In order to start working with the ORM package, you need to enable the it in your /fuel/app/config/config.php file by adding 'orm' to the 'packages' array like this:

'always_load' => array(
	'packages' => array(
		'orm',
	),
),

If you have not included any other FuelPHP package to your project before this point, you may find that the above lines are commented out in your /fuel/app/config/config.php file. Simply uncomment them before adding the 'orm' package.

You do not need to install or download any additional files to work with the ORM package. All the necessary ORM package files are included when you download FuelPHP.

What Next?

Once you have installed the ORM package, you can create your first ORM model. Head over to the Creating Models page for more details.
